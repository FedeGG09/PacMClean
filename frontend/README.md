# Frontend React + FastAPI

## Requisitos
- Node 18+
- Backend FastAPI corriendo en `http://localhost:8000` (o el que definas)

## Instalación
```bash
npm install
```

## Configuración
Crear `.env` en la raíz del frontend:

```env
VITE_API_URL=http://localhost:8000
```

## Ejecución
```bash
npm run dev
```

## Build
```bash
npm run build
```
